*work in progress*

This was my first major python project. It was developed solely by me from scratch. It took me over 6 months to "complete". There are a lot more features that I would like to add in the future, including a more modern UI and more complete integration with other software, but for now it is functional. It is still a work in progress, but it is something I use daily at my current job to assist with processing orders.

The major modules used in this project are:

  • tkinter - UI
  • sqlite - Database Management
  • werkzeug - Password Hashing
  • gspread - Google Sheets Integration

The main features of this program are:

  • Uses sqlite database for for information storage
  • Google Sheets integration to save and upload information
  • Add/Edit/Remove Items and Locations
  • Check history of Items and Locations
  • Create containers to receive items to locations
  • "Cycle Count" to cycle through locations and update quantities
  • Order screen lets you generate a simulated order that will give you an estimated weight and pallet count
  • Add/Edit/Remove users
  • View a log of all changes made
